# psm-webflow-assets

Image assets for the PSM Webflow site — blog thumbnails and case study graphics.

![Assets](https://img.shields.io/badge/type-Image%20Assets-lightgrey)

## Overview

Optimized PNG and JPEG assets used by the PSM Webflow marketing site. Includes blog post thumbnails and case study cover images.

## Naming Convention

- `blog-{slug}_thumb.jpeg` — Blog post thumbnail
- `blog-{slug}.png` — Blog post full image
- `case-{slug}_thumb.jpeg` — Case study thumbnail
- `case-{slug}.png` — Case study full image
